食物主題

先抓方向，餐廳名之後再定。
石垣牛燒肉

放在第一晚。熱門店建議預約，不現場賭運氣。

海人居酒屋

放在 Manta City 當晚，晚餐別排太晚，後面可能接觀星。

在地料理

宮古島先以宮古そば、食堂定食、超市熟食為主，價格最便宜取勝。

公設市場

一定排漢堡跟冰，適合 Day 1 或 Day 2 下午。

竹富島食堂

跳島日中午吃八重山そば，當天上島後先確認營業。

潛水後晚餐

Day 4 不跑遠，便利商店 / 超市 / 近飯店食堂即可，洗澡、休息最重要。

宮古島住宿與飲食：便宜優先

原則：價格最便宜取勝；飯店和餐廳之後依實際日期再比價。

住宿 優先住平良市區

先找平良市區、平良港附近或主要道路旁的商務旅館 / 公寓式旅館。吃飯、超市、藥妝、租車點都比較集中，晚上不用多花交通費。

目前傾向：不追海景。乾淨、可停車、價格低、離晚餐近，比漂亮飯店更重要。

候選 先比這幾類

Mr. Kinjo 系列、公寓式旅館、Hotel Peace Island、Hotel Rising Sun、Hotel Torifito 這類商務 / 平價型住宿先列入比價。

篩選順序：總價最低 → 可免費取消 → 有停車 → 離平良晚餐區近 → 評分不要太低。

飲食 超市與食堂優先

宮古 Day 3、Day 4 不把錢花在餐廳氣氛上。午餐找宮古そば / 食堂定食；晚餐可用超市熟食、便當、簡單居酒屋或近飯店食堂解決。

便宜路線：サンエー、かねひで、マックスバリュ買熟食 / 飲料 / 隔天早餐，最穩也最省。

吃法 宮古兩晚安排

Day 3 晚餐吃平良市區便宜食堂或宮古そば；Day 4 潛水後不跑遠，飯店附近或超市買回去吃。

目前傾向：宮古島不排昂貴晚餐，把預算留給青之洞窟 + 八重干瀨浮潛、租車與石垣牛。

參考方向：Expedia / Tripadvisor 的平價住宿整理多指向平良市區與平價商務旅館；宮古島在地資訊也提到サンエー、かねひで、マックスバリュ等超市可作為省餐費補給點。實際店名與價格需用入住日期再查一次。

宮古海洋活動候選

目前優先：Day 3 南瓜鐘乳洞候選；Day 4 青之洞窟 + 八重干瀨浮潛。

Day 3 南瓜鐘乳洞 + 海上獨木舟

go2ishigaki 方案，9,900 日圓 / 人。探索 Pumpkin Hall 南瓜鐘乳洞，包含游泳進洞、洞內探險、海上獨木舟，可安排接送但需提前告知。

目前傾向：可放 Day 3 下午。它比純陸地景點更有記憶點，但會受潮位、風浪與集合時間影響。

Day 4 青之洞窟 + 八重干瀨浮潛

go2ishigaki 方案，18,500 日圓 / 人。08:30 集合、13:45 解散；青之洞窟後前往八重干瀨，八重干瀨安排 3 個浮潛點，每點約 25-30 分鐘。

目前傾向：Day 4 優先放這個。玩得夠滿、比純八重干瀨浮潛完整，也比 3 dives 方案對體力和預算友善。

南瓜洞 預約與限制

線上預約金 1,500 日圓 / 人，活動後現金支付 8,400 日圓 / 人。對象 7-59 歲；心臟病、高血壓、孕婦等不適合參加。

取消規則偏硬：出發日前 7 日內個人因素取消，可能補收全額旅費；風浪因素取消則退費。

注意 預約與付款

線上預約金 2,500 日圓 / 人，另有 10,000 日圓 / 人保證金；活動後現金支付尾款。付款結構要在下訂前再確認一次。

取消規則偏硬：出發日前 7 日內個人因素取消，可能沒收保證金並補收全額旅費。

交通 自駕前往伊良部

無接送，需自行租車到宮古島市伊良部集合。預約後會提供 map code 和 Google Maps 地點。

前一天先買好船上午餐、飲料、防曬與毛巾；當天不要排太早的其他活動。

備案 3 dives 改進階選項

如果有人仍想要氣瓶潛水，可以另找八重干瀨 3 boat dives 作為進階備案；但目前公開頁主線改以青洞 + 八重干瀨浮潛為優先。

理由：你們已經有石垣 Manta City，宮古這天改成完整浮潛會比較穩、比較不累，也保留八重干瀨核心。

待討論事項

旅伴可以先看「目前傾向」，再一起改。

交通 騎摩托車還是租車？

需要決定石垣、宮古兩邊的主要移動方式。

目前傾向：租車。6 月底 / 7 月初很熱，也可能遇到午後雨；租車比較能放裝備、躲太陽，宮古跑大橋和海灘也更舒服。

水上活動 潛水和水上活動會太多嗎？

目前有 Day 2 Manta City，以及 Day 4 宮古青之洞窟 + 八重干瀨浮潛。

目前傾向：可以。Day 3 是陸地緩衝日，Day 4 晚上留宮古休息，不急著搬家。

探險 要安排西表島嗎？

西表島適合紅樹林、瀑布、獨木舟，但它是一個重型一日行程。

目前傾向：先不加。若加入西表，可能要拿掉竹富 + 小濱輕跳島，或壓縮石垣陸地文化日。

Day 1 抵達後到 check-in 前怎麼過？

10:00 可放行李，但 16:00 才能進房。

目前傾向：咖啡、公設市場、室內手作。沒力氣就只休息，不硬排。

夜間 石垣觀星 tour 要不要排？

星空很吃月相和天氣，滿月附近銀河效果會降低。

目前傾向：保留 Day 2 晚上彈性，接近日期再看天氣和月相。

海洋日 青之洞窟 + 八重干瀨是否定案？

目前看到 go2ishigaki 的 18,500 日圓方案：青之洞窟 + 八重干瀨 3 個浮潛點，08:30 集合、13:45 解散。

目前傾向：優先。3 dives 改成進階備案，不再當主線。

跳島 竹富 + 小濱怎麼玩？

這天不下水，以輕鬆陸地跳島為主。

目前傾向：上午竹富騎腳踏車或水牛車，下午小濱電動腳踏車。船班以回石垣港轉乘為原則。

餐廳 哪些餐廳要先訂？

石垣牛燒肉、海人居酒屋、宮古在地料理都還能細選。

目前傾向：石垣牛和熱門海鮮居酒屋優先預約；其他餐廳等航班、住宿位置確定後再鎖。
