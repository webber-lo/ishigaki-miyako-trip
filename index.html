遺憾 滿月不利銀河

旅人回饋提到唯一遺憾是沒有追到銀河，因為滿月月亮太亮。

這趟觀星收斂到 7/2 晚上。當晚月亮約 21:25 升起，建議抓 20:15-21:20 的月出前空窗；如果天氣不理想，改市區咖啡調酒 / 居酒屋 / 早睡。

吸收 石垣市區日很有價值

咖啡調酒、Gelato、居酒屋、超市、藥妝店可以組成很舒服的市區閒晃路線。

這可以補進 Day 2 下午與 Day 8 回程前，不必把每一天都塞成重型景點。

吸收 石垣陸遊自駕線

參考 The BREAKFAST HOTEL PORTO

旅人回饋：平均一晚不到 3,000，不含早餐；走路約 5 分鐘到觀光區；有廚房、洗衣機、冰箱、微波爐；有 happy hour 酒類暢飲。

可信度先放參考。優點是交通方便、價格中等、房間不算小；缺點是隔音普通、有潮濕味。若石垣住宿還沒鎖，可列入比價，但不直接改現行住宿。

西線 川平灣・夕陽・螢火蟲

市區 / 飯店 → 川平灣 → 沙翁候選 → 南の島のどうぶつ工房イルモ → 御神崎或底地海灘夕陽 → 八重山螢火蟲 tour。

適合 Day 1 抵達後有體力，或 Day 6 石垣緩衝日。若滿月太亮，螢火蟲 / 星空期待值降低。

北線 展望台・平久保・海岸咖啡

市區 → 知念商會早餐 → 玉取崎展望台 → 平久保崎燈塔 / 平久保海邊 → 海岸咖啡廳 → 唐吉訶德 / 思夢樂 → 市區晚餐。

適合 Manta City 取消或多出石垣陸地日。不自駕時點很散，建議包車 / 觀光計程車，或只選 2-3 點。猴子樂園與飛行傘先不放主線。

自駕 北線 + 西線大滿貫

9:00 借車 → 知念商會 → 平久保崎燈塔 → 玉取崎展望台 → 川平灣 → 川平公園茶屋 → 高嶺酒造所買酒 → 市區補貨 → Mirumiru → Fusaki Beach 夕陽 → 20:00 前還車。

如果真的租車，這是目前最像「一天玩完整座石垣」的版本。缺點是點很多，夏天會熱；建議保留刪點彈性，先守住川平灣、Mirumiru、Fusaki 夕陽。

市區 吃喝購物懶人線

公設市場 / Euglena Mall → Hau Tree Gelato → Coral Tree Cafe → KLATCH COFFEE 或 FUSHI COFFEE ROASTERS → Waizu → 八重山牛奶屋 / 8days → Lumi Umi → Hitoshi Ishiganto 或 一魚一會。

適合 Day 2 Manta City 後、雨天、太熱、或 Day 8 回程前。這條線是最不怕天氣的安全牌。

早餐 早開店與咖啡甜點

早開：SEVEN BELLS、石垣島のおにぎり屋 / niginigi、KLATCH COFFEE、Asakara Hirugohan。甜點：Hau Tree Gelato、Mirumiru Hompo Honten。

潛水日用外帶飯糰或麵包；市區慢早晨選 Asakara。Mirumiru 本店在西線，適合川平灣或夕陽路線，不要硬塞進純市區散步。

午餐 在地料理與換口味

あらだ家：石垣島料理、老屋。ウシのカドデ：牛湯底中華そば。Travelers' cafe 朔：咖哩與車海老カレー。

這三家都適合放在市區東側或機場方向前後。吃膩燒肉、居酒屋時，咖哩和拉麵會是很好的換氣口。

晚餐 石垣夜間吃法

正式燒肉：Sumibi-Yakiniku Ishigakian。海鮮居酒屋：Hitoshi Ishiganto。飯後放鬆：Lumi Umi 咖啡調酒。

不要同晚硬塞太多。石垣牛放第一晚，Hitoshi 放海洋日或收尾晚，Lumi Umi 當 7/2 觀星取消備案最漂亮。

已確認 6/27 石垣潛水

Marine shop Jiyujin Ishigaki：Fun diving 3 dives 23,000 円；BCD + regulator 租借 3,500 円。體驗潛水 + 浮潛 20,000 円，追加潛水 6,000 円。

Fin、mask、wetsuit、weights 已含在潛水價格。午餐不含，要自備。前一天與當天不要喝酒；建議加保旅遊 / 潛水保險。

跳島 竹富 + 小濱

石垣港 → 竹富島電動車 / 水牛車 / 村落 → 竹富島食堂八重山そば → 石垣港轉船 → 小濱島電動腳踏車 → 回石垣市區。

竹富島強烈建議電動車。小濱島視船班和體力決定；不要為了湊兩島把回程船班壓太緊。

宮古 海洋優先，陸地少而準

主線：青之洞窟 + 八重干瀨。宮古陸地與吃喝可分散在 Day 3 抵達後與 Day 5 緩衝日；Day 6 早上再回石垣。

不自駕時不要把宮古大橋、東平安名崎、通池、池間全塞滿。Day 5 只選一條線，作為海洋日後的舒服收尾。

先找平良市區、平良港附近或主要道路旁的商務旅館 / 公寓式旅館。吃飯、超市、藥妝、咖啡和叫車都比較集中，晚上不用多花交通費。

目前傾向：不追海景。乾淨、價格低、離晚餐近、活動可接送或容易叫車，比漂亮飯店更重要。

候選 先比這幾類

Mr. Kinjo 系列、公寓式旅館、Hotel Peace Island、Hotel Rising Sun、Hotel Torifito 這類商務 / 平價型住宿先列入比價。

線上預約金 2,500 日圓 / 人，另有 10,000 日圓 / 人保證金；活動後現金支付尾款。付款結構要在下訂前再確認一次。

取消規則偏硬：出發日前 7 日內個人因素取消，可能沒收保證金並補收全額旅費。

交通 伊良部集合怎麼去？

此方案頁面寫無接送，集合在宮古島市伊良部。若不開車，需要預先安排計程車或觀光計程車，不能當天臨時賭車。

前一天先買好船上午餐、飲料、防曬與毛巾；並請住宿或活動店家協助確認去程與回程叫車。

備案 3 dives 改進階選項

如果有人仍想要氣瓶潛水，可以另找八重干瀨 3 boat dives 作為進階備案；但目前公開頁主線改以青洞 + 八重干瀨浮潛為優先。

理由：你們已經有石垣 Manta City，宮古這天改成完整浮潛會比較穩、比較不累，也保留八重干瀨核心。

非自駕交通解法

使用者傾向：機車不當主交通；若 Day 5 要跑宮古陸地線，可討論租汽車。

宮古 活動接送優先

青之洞窟 + 八重干瀨方案目前頁面寫無接送，伊良部集合需另想辦法。南瓜鐘乳洞若排 Day 5，也要提前確認接送、集合與沖洗。

下訂前先問店家：能否接送、可否協助叫車、活動結束地點是否容易叫計程車。

宮古 計程車 / 觀光計程車

宮古機場與下地島機場官網都有計程車資訊；宮古當地也有觀光計程車業者，可做半日或點到點移動。

Day 3 若想補宮古陸地景點，與其硬排分散景點，可以包 3-4 小時觀光計程車跑幾個重點。

租車 ミツバチGoKart / アイリスレンタカー

3850.jp：MINI コンバーチブル専門的ミツバチGoKartレンタカー，也有一般國產車的アイリスレンタカー與 3 輪バイク。營業 8:30-19:00。

若 Day 5 要跑池間 / 伊良部 / 下地島或東岸線，租汽車比機車合理。MINI 敞篷車偏體驗感，一般車款偏實用；需確認台灣駕照日文譯本、保險、取還車點與停車。

住宿 平良市區更重要

不開車時，住宿位置比海景更重要。平良市區吃飯、超市、咖啡、叫車、集合都比較容易。

住宿篩選要加入：離晚餐區近、可幫忙叫車、是否在活動接送範圍。

陸地日 景點少而準

不自駕就不要把東平安名崎、來間、池間、通池、17END 全塞在同一天。

只選 2-3 個最想看的點，或直接用巴士 / 觀光計程車行程取代自排路線。

機車風險 具體問題 對本行程的影響 建議
駕照 / 法規 台灣駕照在日本通常需搭配日文譯本；機車級距也要符合租借車種，不能只看汽車駕照。 若文件或級距不符，現場可能租不到，或保險理賠有風險。 除非每位騎士都先確認合法文件與可租車種，否則不要把機車排成必要交通。
天氣 / 體感 6 月底 / 7 月初炎熱，午後雨、強日照、濕黏感都會放大疲勞。 Day 3 若下午再騎車跑景點，容易影響 Day 4 青洞 + 八重干瀨體力。 活動日與海上活動前一天不建議靠機車長距離移動。
橋與風 宮古常見景點會跨伊良部大橋、池間大橋、來間大橋；橋上風、橫風、曝曬都比市區明顯。 去 17END、通池、池間島、來間島時，機車舒適度與安全性都下降。 跨橋景點優先用計程車、觀光計程車或套裝行程，不用機車硬跑。
行李 / 裝備 浮潛、毛巾、防曬、換洗衣物、飲水、購物袋都不適合用機車載太多。 Day 4 船上活動前後，身體濕、曬、累，還要騎車會很麻煩。 海洋活動集合點用接送或計程車，機車不放在 Day 4。
酒精 / 晚餐 晚餐若吃燒肉、居酒屋或喝泡盛，機車會直接限制大家。 宮古晚上若住平良市區，步行 / 計程車比機車合理。 晚餐日不租機車；住宿選靠近晚餐區。

結論：機車不當主交通。若 Day 5 想跑多點或跨橋，優先討論租汽車；若只是體驗感，MINI 敞篷車可當宮古陸地日亮點，但需先確認證件、保險、取還車與停車。

待討論事項

旅伴可以先看「目前傾向」，再一起改。

交通 不想開車怎麼移動？

需要把石垣、宮古兩邊改成非自駕策略，特別是宮古的伊良部、南瓜洞、海洋活動集合點。

目前傾向：海洋活動用接送 / 計程車；Day 5 陸地日可比較 3850.jp 租汽車 vs 觀光計程車。景點不要排太散，租車也只選一條線。

水上活動 潛水和水上活動會太多嗎？

Day 2 6/27 Marine shop Jiyujin Ishigaki 已確認：Fun diving 3 dives；同行者有體驗潛水 + 浮潛安排。Day 4 再排宮古青之洞窟 + 八重干瀨浮潛。

目前傾向：可以，但 6/27 下午和晚上要降載。午餐不含需自備；60 歲以上潛水需醫師證明，高血壓、糖尿病、氣胸等狀況不可參加。

取消 6/27 潛水取消規則

天候不佳如颱風取消不收費；3 天前取消不收費；2 天前 30%；前一天 50%；當天 100%。

若店家聞到酒味會要求酒測，超標不得參加且收 100% 取消費。前一晚晚餐要收斂，酒精直接避開。

探險 要安排西表島嗎？

西表島適合紅樹林、瀑布、獨木舟，但它是一個重型一日行程。

目前傾向：先不加。若加入西表，可能要拿掉竹富 + 小濱輕跳島，或壓縮石垣陸地文化日。

Day 1 抵達後到 check-in 前怎麼過？

10:00 可放行李，但 16:00 才能進房。

目前傾向：咖啡、公設市場、室內手作。沒力氣就只休息，不硬排。

夜間 石垣觀星 tour 排哪天？

定在 Day 7，7/2 晚上。月亮約 21:25 升起，正式觀星抓 20:15-21:20 的月出前空窗。

6/27 已有 3ROUND 水上活動，不排觀星。7/2 若天氣或雲量不理想，改 Lumi Umi / 居酒屋 / 早睡。

海洋日 青之洞窟 + 八重干瀨是否定案？

目前看到 go2ishigaki 的 18,500 日圓方案：青之洞窟 + 八重干瀨 3 個浮潛點，08:30 集合、13:45 解散。

目前傾向：優先。3 dives 改成進階備案，不再當主線。

跳島 竹富 + 小濱怎麼玩？

這天不下水，以輕鬆陸地跳島為主。

目前傾向：上午竹富騎腳踏車或水牛車，下午小濱電動腳踏車。船班以回石垣港轉乘為原則。

餐廳 哪些餐廳要先訂？

石垣牛燒肉、海人居酒屋、宮古在地料理都還能細選。

目前傾向：石垣牛和熱門海鮮居酒屋優先預約；其他餐廳等航班、住宿位置確定後再鎖。
