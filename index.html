遺憾 滿月不利銀河

旅人回饋提到唯一遺憾是沒有追到銀河，因為滿月月亮太亮。

這趟觀星收斂到 7/2 晚上。當晚月亮約 21:25 升起，建議抓 20:15-21:20 的月出前空窗；如果天氣不理想，改市區咖啡調酒 / 居酒屋 / 早睡。

吸收 石垣市區日很有價值

咖啡調酒、Gelato、居酒屋、超市、藥妝店可以組成很舒服的市區閒晃路線。

這可以補進 Day 2 下午與 Day 8 回程前，不必把每一天都塞成重型景點。

參考 The BREAKFAST HOTEL PORTO

旅人回饋：平均一晚不到 3,000，不含早餐；走路約 5 分鐘到觀光區；有廚房、洗衣機、冰箱、微波爐；有 happy hour 酒類暢飲。

可信度先放參考。優點是交通方便、價格中等、房間不算小；缺點是隔音普通、有潮濕味。若石垣住宿還沒鎖，可列入比價，但不直接改現行住宿。

自駕 北線 + 西線大滿貫

9:00 借車 → 知念商會 → 平久保崎燈塔 → 玉取崎展望台 → 川平灣 → 川平公園茶屋 → 高嶺酒造所買酒 → 市區補貨 → Mirumiru → Fusaki Beach 夕陽 → 20:00 前還車。

如果真的租車，這是目前最像「一天玩完整座石垣」的版本。缺點是點很多，夏天會熱；建議保留刪點彈性，先守住川平灣、Mirumiru、Fusaki 夕陽。

早餐 早開店與咖啡甜點

潛水日判斷：如果 08:00 左右集合，吃 KLATCH COFFEE；如果更早集合，前一天先買好；如果潛水回來後想補吃，再去 SEVEN BELLS 或知念商會。

niginigi / Asakara 不放 6/27 潛水早晨主線。Mirumiru 本店在西線，適合其他路線，不硬塞進純市區散步。

午餐 在地料理與換口味

あらだ家：石垣島料理、老屋。ウシのカドデ：牛湯底中華そば。Travelers' cafe 朔：咖哩與車海老カレー。

這三家都適合放在市區東側或機場方向前後。吃膩燒肉、居酒屋時，咖哩和拉麵會是很好的換氣口。

晚餐 石垣夜間吃法

正式燒肉：Sumibi-Yakiniku Ishigakian。海鮮居酒屋：Hitoshi Ishiganto。飯後放鬆：Lumi Umi 咖啡調酒。

不要同晚硬塞太多。石垣牛放第一晚，Hitoshi 放海洋日或收尾晚，Lumi Umi 當 7/2 觀星取消備案最漂亮。

已確認 6/27 石垣潛水

Marine shop Jiyujin Ishigaki：Fun diving 3 dives 23,000 円；BCD + regulator 租借 3,500 円。體驗潛水 + 浮潛 20,000 円，追加潛水 6,000 円。

Fin、mask、wetsuit、weights 已含在潛水價格。午餐不含，要自備。前一天與當天不要喝酒；建議加保旅遊 / 潛水保險。

跳島 竹富 + 小濱

石垣港 → 竹富島電動車 / 水牛車 / 村落 → 竹富島食堂八重山そば → 石垣港轉船 → 小濱島電動腳踏車 → 回石垣市區。

竹富島強烈建議電動車。小濱島視船班和體力決定；不要為了湊兩島把回程船班壓太緊。

宮古 海洋優先，陸地少而準

主線：6/29 下午 13:00 宮古島八重干瀨、6/30 上午南瓜洞探險。宮古租車服務這兩個活動集合與回程沖洗。

不要把宮古大橋、東平安名崎、通池、池間全塞滿。已確定活動優先，其他景點只作備案。

宮古島住宿與飲食：便宜優先

原則：價格最便宜取勝；飯店和餐廳之後依實際日期再比價。

住宿 優先住平良市區

先找平良市區、平良港附近或主要道路旁的商務旅館 / 公寓式旅館。吃飯、超市、藥妝、咖啡和叫車都比較集中，晚上不用多花交通費。

目前傾向：不追海景。乾淨、價格低、離晚餐近、活動可接送或容易叫車，比漂亮飯店更重要。

候選 先比這幾類

Mr. Kinjo 系列、公寓式旅館、Hotel Peace Island、Hotel Rising Sun、Hotel Torifito 這類商務 / 平價型住宿先列入比價。

篩選順序：總價最低 → 可免費取消 → 離平良晚餐區近 → 活動接送範圍 / 容易叫車 → 評分不要太低。

飲食 超市與食堂優先

宮古 Day 3、Day 4 不把錢花在餐廳氣氛上。午餐找宮古そば / 食堂定食；晚餐可用超市熟食、便當、簡單居酒屋或近飯店食堂解決。

便宜路線：サンエー、かねひで、マックスバリュ買熟食 / 飲料 / 隔天早餐。甜點小繞路則用 SUSHEMARO、海美来、宮古きび茶屋、シモジスーパー補樂趣。

吃法 宮古兩晚安排

Day 3 晚餐吃平良市區便宜食堂或宮古そば；如果想安排宮古正式晚餐，炭火焼肉 琉宮苑當候選。Day 4 潛水後不跑遠，飯店附近或超市買回去吃。

目前傾向：宮古島以省錢為主，但保留一個正式晚餐彈性；甜點和在地超市作為順路加分，不為單點拉長交通。

參考方向：Expedia / Tripadvisor 的平價住宿整理多指向平良市區與平價商務旅館；宮古島在地資訊也提到サンエー、かねひで、マックスバリュ等超市可作為省餐費補給點。實際店名與價格需用入住日期再查一次。

宮古實戰提醒

旅人回饋，先當現場執行注意事項；不是全部都要改主線，但會影響租車、補給、海灘與行李準備。

通訊 e-SIM 分散風險

建議不要三個人都買同一家。回饋是 KDDI 與 docomo 覆蓋感受不同，docomo 在部分地點可能會掛。

目前吸收：至少一人買 KDDI，另一人可用不同系統，避免宮古跨島或海邊時全員沒訊號。

租車 加油與還車

還車前通常要到指定加油站加滿油並拿證明。市區加油站可能較便宜，可先在市區補一些，再到指定站補滿。

可記 apollostation みなみ給油所 / （有）豊見山石油為參考；實際仍以租車店指定規則為準。

超市 退稅與補給

超市退稅時間可能只到晚上八點或九點，建議提早去。Day 3 / Day 4 補給不要壓到太晚。

目前吸收：San-A / 超市補給排在 Day 3 下午與 Day 4 傍晚，不把退稅和隔天早餐留到深夜。

海灘 陽傘與裝備

白天海邊若要玩 3-4 小時以上，租陽傘比硬曬合理；回饋價格約一天一傘 3,000 円。

淺灘建議自己帶面鏡或買好面鏡；救生衣為安全仍建議租。溯溪鞋比蛙鞋更適合淺灘走路，不要赤腳。

浮潛 先跟教練玩一次

旅人建議第二天先安排有教練的浮潛，熟悉水性和場地後，後面才考慮淺灘自己玩。

我們已有 Day 4 青洞 + 八重干瀨，符合先跟教練的邏輯；之後若加新城海岸，才考慮淺灘自主玩水。

新城 早去才舒服

新城海岸真的要早點去，停車費回饋約 2,000 円，偏盤但熱門。

目前不放主線。若租車日臨時加海灘，只能排早上，不要下午熱到崩潰才去。

吃喝 國仲商店與池間 Gelato

國仲商店要早點去，麵包可能賣完。池間島 Gelato 下方海灘有很多寄居蟹，可多留一點時間。

兩者都不硬塞主線；若 Day 5 租車後有空想北線小繞路，才把池間 Gelato 當加碼點。

若租車經過雪鹽方向再買，不為袋裝薯條單獨拉一條線。

住宿 硬水與飲水

宮古島水質偏硬，沒住大飯店可先問飯店是否有濾水系統；喝不慣就便利商店買軟水。

目前吸收：Day 3 超市補給加買軟水，海洋日與南瓜洞前都先買足。
